========
|  GIT |
========
- Clone a repository (project)
$ git clone

- Stage a file to commit
$ git add

- Commit staged file(s)
$ git commit -m "<commit-message>"

- Push changes (committed files) to repository
$ git push

- Pull changes from repository to local project
$ git pull

- Show changes made in a specific commit
$ git show <commit-hash>

- Show changes between local and origin respository
$ git diff

=======
| VIM |
=======
NOTE: most Vim commands can be sequentially applied

- Write a file
$ :w

- Quit a file
$ :q

- Enter Insert mode
$ i


- Exit Insert mode
$ ESC

- Move up
$ k

- Move down
$ j

- Move right
$ l

- Move left
$ h

===============
| UBUNTU BASH |
===============
- Open a file from terminal using vim
$ vim <file-name>

- Change directory
$ cd

- Show files and directories in current path
$ ls

- Clear screen
$ clear

- Exit terminal
$ exit

- Kill process
$ <CTRL> C

- Make a directory
$ mkdir <directory-name>

- Remove a file
$ rm <file-name>

- Remove a directory
$ rm -rd <directory-name>
